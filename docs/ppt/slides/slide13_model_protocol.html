物模型与协议设计

Aquarium 属性服务（13 字段）

  • 环境类：temperature / ph / tds / turbidity / water_level
  • 执行器：heater / pump_in / pump_out
  • 运行模式：auto_mode / feeding_in_progress
  • 投喂：feed_countdown（周期或一次性）
  • 告警：alarm_level / alarm_muted

命令与 Topic

  • control：加热/泵/投喂/静音/目标温度
  • set_thresholds：温度/pH/TDS/浊度/水位阈值
  • set_config：Wi-Fi + 校准系数
  • 上报 Topic：$oc/devices/{id}/sys/properties/report
  • 命令 Topic：$oc/devices/{id}/sys/commands/#

字段与接口保持设备端与应用端一致，避免“字段漂移”导致的控制失效。
