传感器/执行器接口映射

传感器接口（ADC/单总线）

  • 温度 DS18B20：PA8（单总线，4.7kΩ 上拉）
  • pH 传感器：PA0（A0）
  • TDS 传感器：PA1（A1）
  • 浊度传感器：PA4（A2）
  • 水位传感器：PB0（A3，分压后 0–3.3V）

执行器与通信接口

  • 进水泵继电器：PB5（D4）
  • 排水泵继电器：PB4（D5，禁用 JTAG）
  • 加热棒继电器：PB10（D6）
  • 舵机投饵：PC7（TIM3_CH2）
  • ESP32 AT：PA2/PA3（USART2）

关键引脚与供电注意事项已在 docs/TECH.MD 与 Wiring_Guide.md 中详细说明。
