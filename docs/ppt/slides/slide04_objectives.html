建设目标与关键指标

功能目标

  • 多传感器实时监测：温度 / pH / TDS / 浊度 / 水位
  • 执行器控制：加热棒、进水泵、排水泵、投饵舵机
  • OLED 本地显示 + 蜂鸣器/LED 分级告警
  • 支持自动/手动模式与阈值配置
  • IoTDA 云端注册、影子上报、命令下发
  • HarmonyOS App 可视化仪表盘与远程控制

非功能指标

  • 稳定性：设备连续运行 12 小时无异常复位
  • 可靠性：传感器失效自动容错与安全值输出
  • 易用性：AP 配网 ≤ 5 分钟完成
  • 性能：影子刷新 1 秒内更新
  • 演示约定：AK/SK 签名 + 固定内置参数（便于快速联调）
  • 可扩展性：新增传感器仅需扩展物模型

目标对齐 PRD 与验收标准，确保“设备—云—应用”闭环可演示。
